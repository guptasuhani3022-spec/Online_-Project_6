# 📱 Mobile User Segmentation System

A professional Machine Learning project built using **Python**, **Streamlit**, and **K-Means Clustering** to segment mobile users based on their usage behavior.

---

## 🚀 Overview

This project analyzes mobile user data and automatically groups users into different categories using the **K-Means Clustering Algorithm**.

The application provides an interactive dashboard where users can:

- 📊 Explore the dataset
- 📈 View statistical analysis
- 📉 Visualize user behavior
- 🤖 Predict user category
- 📥 Download prediction reports

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/mobile-user-segmentation.git
```

Move to project folder

```bash
cd mobile-user-segmentation
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
